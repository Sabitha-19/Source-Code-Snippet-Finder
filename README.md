# Source Code Snippet Finder

## Objective
To build a Flask-based web application that retrieves relevant Python code snippets using Smoothed TF-IDF ranking.

## Features
- Keyword-based search
- TF-IDF ranking
- Snippet preview
- Copy-to-clipboard functionality

## Technologies Used
- Python
- Flask
- HTML
- Git & GitHub

## Installation
pip install -r requirements.txt

## Run the Application
python app.py
Open http://127.0.0.1:5000
